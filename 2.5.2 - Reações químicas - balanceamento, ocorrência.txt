**Questão 03**

Para contribuir com a captação de CO₂ atmosférico, uma unidade industrial chilena anunciou a produção de gasolina sintética. Como o principal componente da gasolina é o isooctano (C₈H₁₈), o processo industrial empregado se baseou na reação química representada pela seguinte equação não balanceada:

**CO₂ (g) + H₂ (g) ——→ C₈H₁₈ (ℓ) + H₂O (ℓ)**

Observe na tabela as entalpias-padrão de formação das substâncias que participam dessa reação:

**Tabela de entalpias-padrão de formação:**

| SUBSTÂNCIA | ENTALPIA-PADRÃO DE FORMAÇÃO (kJ/mol) |
|------------|--------------------------------------|
| CO₂ (g) | -394 |
| H₂ (g) | 0 |
| C₈H₁₈ (ℓ) | -209 |
| H₂O (ℓ) | -286 |

Após balancear a equação, determine a razão entre o número de mols de CO₂ e H₂. Calcule, ainda, a variação de entalpia da reação, em quilojoules.

---

**Gabarito Oficial**

**Equação balanceada:** 8 CO₂ + 25 H₂ → C₈H₁₈ + 16 H₂O

**Razão:** CO₂/H₂ = 8/25

**Cálculo da variação de entalpia:**
```
ΔH° = (ΔH°f C₈H₁₈) + (16 × ΔH°f H₂O) - (8 × ΔH°f CO₂) - (25 × ΔH°f H₂)
ΔH° = -209 - (16 × 286) + (8 × 394) = -209 - 4576 + 3152 = -1633 kJ
```

---

**Contexto para LLM:**
Esta questão integra conceitos de termoquímica com aplicações industriais sustentáveis. Os temas centrais incluem:

- **Balanceamento químico:** Equações complexas com múltiplos produtos
- **Termoquímica:** Cálculo de variação de entalpia usando entalpias de formação
- **Sustentabilidade:** Captura e conversão de CO₂ atmosférico
- **Química industrial:** Produção de combustíveis sintéticos
- **Estequiometria:** Proporções molares em reações químicas

**Conceitos-chave para o LLM:**
- **Entalpia de formação (ΔH°f):** Energia para formar 1 mol de substância a partir dos elementos
- **Lei de Hess:** ΔH°reação = Σ ΔH°f(produtos) - Σ ΔH°f(reagentes)
- **Balanceamento:** Conservação de átomos de cada elemento
- **Isooctano:** C₈H₁₈, componente principal da gasolina
- **Reação exotérmica:** ΔH° negativo indica liberação de energia

**Dados importantes da tabela:**
- **CO₂ (g):** ΔH°f = -394 kJ/mol
- **H₂ (g):** ΔH°f = 0 kJ/mol (elemento no estado padrão)
- **C₈H₁₈ (ℓ):** ΔH°f = -209 kJ/mol
- **H₂O (ℓ):** ΔH°f = -286 kJ/mol

**Processo de balanceamento:**
1. **Carbono:** 8 CO₂ → C₈H₁₈ (8 carbonos)
2. **Hidrogênio:** 18 H no produto + 32 H na água = 50 H total → 25 H₂
3. **Oxigênio:** 16 O₂ do CO₂ = 16 O na água
4. **Equação balanceada:** 8 CO₂ + 25 H₂ → C₈H₁₈ + 16 H₂O

**Cálculo termoquímico detalhado:**
```
ΔH°reação = [ΔH°f(C₈H₁₈) + 16 × ΔH°f(H₂O)] - [8 × ΔH°f(CO₂) + 25 × ΔH°f(H₂)]

Produtos:
- C₈H₁₈: 1 × (-209) = -209 kJ
- H₂O: 16 × (-286) = -4576 kJ
- Total produtos: -4785 kJ

Reagentes:
- CO₂: 8 × (-394) = -3152 kJ
- H₂: 25 × (0) = 0 kJ
- Total reagentes: -3152 kJ

ΔH°reação = -4785 - (-3152) = -1633 kJ
```

**Aspectos ambientais e industriais:**
- **Captura de CO₂:** Conversão de gás do efeito estufa em combustível
- **Economia circular:** Reutilização de resíduos atmosféricos
- **Energia renovável:** H₂ pode ser produzido por eletrólise sustentável
- **Combustíveis sintéticos:** Alternativa aos combustíveis fósseis

**Significado do resultado:**
- **Razão 8:25:** Para cada 8 mols de CO₂, são necessários 25 mols de H₂
- **ΔH° = -1633 kJ:** Reação altamente exotérmica (libera muita energia)
- **Viabilidade:** Reação termodinamicamente favorável

**Processo de resolução:**
1. **Balancear a equação:** Começar pelo elemento mais complexo (C)
2. **Calcular a razão:** Usar coeficientes da equação balanceada
3. **Aplicar Lei de Hess:** Somar entalpias de produtos e subtrair reagentes
4. **Verificar sinais:** Negativo indica processo exotérmico

Nível de dificuldade: Médio-Alto
Área: Termoquímica e Estequiometria
Habilidades avaliadas: Balanceamento de equações complexas, cálculos termoquímicos, aplicação da Lei de Hess, interpretação de dados tabulares, contextualização com sustentabilidade ambiental.

Esta questão integra conceitos de termoquímica, Lei de Hess e processos industriais:

1. **Reação industrial estudada**:
   - **Processo**: Produção de gás hidrogênio

   - **Importância**: Fonte industrial de H₂

   - **Reação 1**: H₂ + ½ O₂ → H₂O (g), ΔH° = -242,0 kJ
   - **Reação 2**: C + O₂ → CO₂ (g), ΔH° = -393,5 kJ  
   - **Reação 3**: O₂ + 2 CO → 2 CO₂ (g), ΔH° = -477,0 kJ
   - **Condições**: 25°C, 1 atm (condições padrão)

3. **Aplicação da Lei de Hess**:
   - **Princípio**: ΔH independe do caminho da reação
   - **Método**: Manipulação algébrica de equações
   - **Objetivo**: Obter a equação desejada

4. **Manipulação das equações**:
   
   **Equação 1 (invertida)**:
   - Original: H₂ + ½ O₂ → H₂O, ΔH° = -242,0 kJ
   - Invertida: H₂O → H₂ + ½ O₂, ΔH° = +242,0 kJ
   
   **Equação 2 (inalterada)**:
   - C + O₂ → CO₂, ΔH° = -393,5 kJ
   
   **Equação 3 (invertida e dividida por 2)**:
   - Original: O₂ + 2 CO → 2 CO₂, ΔH° = -477,0 kJ
   - Invertida: 2 CO₂ → O₂ + 2 CO, ΔH° = +477,0 kJ
   - Dividida por 2: CO₂ → ½ O₂ + CO, ΔH° = +238,5 kJ

5. **Soma das equações manipuladas**:
   ```
   H₂O (g) → H₂ (g) + ½ O₂ (g)           ΔH° = +242,0 kJ
   C (s) + O₂ (g) → CO₂ (g)               ΔH° = -393,5 kJ  
   CO₂ (g) → ½ O₂ (g) + CO (g)           ΔH° = +238,5 kJ
   ________________________________________________
   H₂O (g) + C (s) → CO (g) + H₂ (g)     ΔH° = +87,0 kJ
   ```

6. **Verificação do cancelamento**:
   - **O₂**: ½ O₂ + O₂ - ½ O₂ = O₂ (cancela)
   - **CO₂**: +CO₂ - CO₂ = 0 (cancela)
   - **Resultado**: Apenas reagentes e produtos desejados

7. **Cálculo da energia para 1 kg de H₂**:
   
   **Interpretação do ΔH°**:
   - ΔH° = +87 kJ para 1 mol de H₂ produzido
   - Massa molar do H₂ = 2 g/mol
   - Portanto: 87 kJ necessários para produzir 2 g de H₂
   
   **Proporção**:
   - 87 kJ → 2 g de H₂
   - X → 1.000 g de H₂
   - X = (87 × 1.000) / 2 = 43.500 kJ = 4,35 × 10⁴ kJ

8. **Identificação do agente redutor**:
   - **Análise da reação**: H₂O + C → CO + H₂
   - **Estados de oxidação**:
     - C: 0 → +2 (perde elétrons, oxidação)
     - H: +1 → 0 (ganha elétrons, redução)
   - **Agente redutor**: Carbono (provoca redução do H⁺)

9. **Estados de oxidação detalhados**:
   ```
   H₂O + C → CO + H₂
   +1-2   0   +2-2  0
   ```
   - **Carbono**: 0 → +2 (oxidação)
   - **Hidrogênio**: +1 → 0 (redução)

10. **Aspectos energéticos**:
    - **ΔH° > 0**: Reação endotérmica
    - **Necessidade**: Fornecimento de energia (aquecimento)
    - **Condições industriais**: Altas temperaturas (800-1000°C)

11. **Processo industrial - Gás d'água**:
    - **Matérias-primas**: Vapor d'água + carvão/coque

    ```
    CO + H₂O → CO₂ + H₂ (reação de deslocamento)
    ```
    - **Objetivo**: Aumentar produção de H₂
    - **Resultado**: Conversão de CO em mais H₂

13. **Conceitos avaliados**:
    - **Lei de Hess**: Manipulação de equações termoquímicas
    - **Termoquímica**: Cálculos de variação de entalpia
    - **Estequiometria**: Relações mol-massa
    - **Oxidação-redução**: Identificação de agentes

14. **Verificação dimensional**:
    ```
    kJ/mol × mol = kJ
    (87 kJ/2g) × 1000g = 43.500 kJ = 4,35 × 10⁴ kJ
    ```

    - **Produção de H₂**: Matéria-prima para diversos processos

    - **Refinarias**: Hidrocraqueamento, dessulfurização
